# ROS2 Mobile Robot System

## Overview
This project is the second-stage extension of my ROS2 navigation learning journey.

The goal is to move from simulation-based navigation toward a real robot system by connecting ROS2 with embedded hardware.

## Current Focus
Phase 1 focuses on building the communication bridge between ROS2 and Arduino Uno R3.

Target:
ROS2 → Serial Communication → Arduino → Basic Response
