# Day7 Log - STM32 PWM Control and Motor Driver Integration

## Summary

Today I transitioned from GPIO-based control to PWM-based control and initiated integration with the TB6612 motor driver.

---

## Work Done

### 1. PWM Configuration

* Configured TIM2 Channel 1 as PWM output using CubeMX
* Assigned pin: PA0 (TIM2_CH1)
* Parameters:

  * Prescaler = 83
  * Period = 999
* Generated initialization code

### 2. PWM Verification

* Started PWM using HAL_TIM_PWM_Start
* Controlled duty cycle using __HAL_TIM_SET_COMPARE
* Connected external LED to PA0

Result:

* LED brightness changes with duty cycle
* PWM output verified functional

---

### 3. Continuous PWM Control

* Implemented gradual duty cycle variation using for-loops
* Adjusted step size and delay to achieve smooth transitions

Result:

* Breathing LED effect achieved
* Improved understanding of continuous control via PWM

---

### 4. Control Model Upgrade

Established distinction between control types:

* GPIO: discrete ON/OFF control
* PWM: continuous intensity/energy control

System abstraction:

STM32 → PWM signal → output behavior

---

### 5. TB6612 Integration (Initial Attempt)

#### Pin Mapping

* PA0 → PWMA
* PA1 → AIN1
* PA4 → AIN2
* PA6 → STBY

#### Motor Interface

* Identified M and P as motor terminals
* Ignored encoder pins (G, A, B, V) for this stage

#### Wiring

* Control signals connected
* Motor connected to AO1/AO2
* Power connections established (VCC, VM, GND)

---

## Issues Encountered

* Motor did not rotate after full setup

Possible causes:

* Unstable or incorrect GND connection
* VM (motor supply) not properly delivering power
* USB power insufficient for motor startup
* STBY signal not effectively enabled
* Loose or incorrect wiring

---

## Key Learnings

* PWM is required for continuous control in embedded systems
* STM32 outputs digital signals; analog behavior is achieved via PWM
* Separation of system layers:

  * Control (STM32)
  * Power (driver)
  * Execution (motor)
* Hardware debugging requires systematic verification
