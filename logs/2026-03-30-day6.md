# Day6.75 Log - STM32 GPIO Input Control

## Summary
Today I extended the STM32 project from simple LED blinking to basic input-output control.

## Work Done

### 1. Repository Management
- Moved STM32 project into Linux repository under `firmware/stm32/f401`
- Cleaned build artifacts (Debug folder)
- Established proper firmware project structure

### 2. GPIO Input Configuration
- Configured PC13 as GPIO input with Pull-up
- Verified board USER button mapping

### 3. Control Logic Implementation
- Implemented button-controlled LED behavior:
  - Press → LED ON
  - Release → LED OFF

### 4. Advanced Logic (State Toggle)
- Implemented edge-triggered logic:
  - Press once → toggle LED state
- Introduced:
  - state variable (`led_state`)
  - last state tracking (`last_button_state`)
  - basic debounce (HAL_Delay)

## Key Learnings
