# Day11 Log - Single Motor Bring-up

## Summary
After ~3 days of debugging, STM32 + TB6612 + DC motor control chain is fully working.

Now supports:
- forward / reverse
- PWM speed control
- stop

Single motor bring-up completed.

---

## Key Issues Found

### 1. STBY mapping error
- PA6 ≠ D6  
- Correct: PA6 → D12  

→ TB6612 was not enabled before fix

### 2. AIN2 mapping error
- PA4 ≠ A4  
- Correct: PA4 → A2  

→ reverse control failed before fix

---

## Final Wiring Mapping
- PA0 → A0 → PWMA  
- PA1 → A1 → AIN1  
- PA4 → A2 → AIN2  
- PA6 → D12 → STBY  

---

## Verified Control Logic

Direction:
- AIN1 = 1, AIN2 = 0 → forward  
- AIN1 = 0, AIN2 = 1 → reverse  

PWM:
- 300 → low speed  
- 800 → high speed  

---

## Result
Motor can now:
- forward low/high  
- reverse low/high  
- stop  

Control chain verified:
STM32 → TB6612 → Motor

---

## Reflection
Main issues were not algorithmic, but:
- pin mapping confusion (PAx vs Ax/Dx)  
- wiring mistakes  

Typical embedded debugging case.
